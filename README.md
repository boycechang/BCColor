Adding BCColor to your project
====================================

#### Requirements

* ARC only; iOS 7.0+

#### Get it as: 
##### 1) source files

1. Download the BCColor repository as a zip file or clone it
2. Copy the BCColor files into your Xcode project

##### 2) via Cocoa pods
